# Authenticity Validator

An AI-powered document analysis system that detects:

- 📄 Plagiarism Score
- 🤖 AI Generated Probability
- 🔍 Text Similarity using TF-IDF
- 📂 File Upload Support (PDF & DOCX)

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer

### Frontend
- React.js
- Axios

## Features

- User Authentication
- Document Upload
- AI Probability Detection
- Plagiarism Similarity Index
- Text Extraction from PDF & DOCX

## How It Works

1. User uploads document
2. Text is extracted
3. Text is preprocessed
4. TF-IDF similarity calculated
5. AI probability calculated
6. Results returned and stored
